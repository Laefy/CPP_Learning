Visual Studio Code

Pour l’IDE, nous vous conseillons d’utiliser Visual Studio Code, car celui-ci est disponible sur toutes les plateformes (contrairement à Visual Studio tout court), qu’il est très bien documenté et que vous pourrez trouvez de nombreux tutoriels en ligne pour apprendre à l’utiliser.
Vous trouverez ci-dessous les instructions permettant d’installer VSCode et un compilateur (g++ pour Windows et Linux, clang pour MacOS), de compiler un programme et de l’exécuter depuis VSCode.

Windows

  1. Installez VSCode en suivant les instructions du paragraphe Installation.
    Si l’installeur vous propose d’installer git, vous pouvez acceptez, cela vous fera gagner du temps pour la suite.
  2. Installez le compilateur g++ en suivant les étapes 3 et 4 de la section Prerequisites.
  3. Installez ensuite l’extension C/C++ (Microsoft).
    • Ouvrez l’application VSCode.
    • Allez dans View > Extensions.
    • Recherchez l’extension C/C++.
    • Cliquez sur le bouton Install.
  4. Testez votre installation soit en suivant cette vidéo, soit suivant les instructions des sections suivantes :
    • Create Hello World.
    • Build helloworld.cpp.
    • Debug helloworld.cpp.

Linux

  1. Installez VSCode en suivant les instructions du paragraphe Installation.
  2. Installez le compilateur g++ en suivant les instructions de la section Ensure GCC is installed.
  3. Installez ensuite l’extension C/C++ (Microsoft).
    • Ouvrez l’application VSCode.
    • Allez dans View > Extensions.
    • Recherchez l’extension C/C++.
    • Cliquez sur le bouton Install.
  4. Testez votre installation soit en suivant cette vidéo, soit suivant les instructions des sections suivantes :
    • Create Hello World.
    • Build helloworld.cpp.
    • Debug helloworld.cpp.
